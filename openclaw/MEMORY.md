# OpenClaw 长期记忆

## 项目概述

### 项目名称
研磨设备B2B独立站 (PrecisionGrind)

### 项目目标
建立一个专业的B2B电子商务平台，销售研磨设备及相关产品，包括研磨罐、研磨介质、研磨设备和配件。

### 目标市场
- 实验室和研究机构
- 工业制造企业
- 电子、制药、化工、航空航天等行业

## 业务知识

### 产品类别
1. **研磨罐 (Grinding Jars)**
   - 陶瓷类: 氧化铝、氧化锆、氮化硅
   - 金属类: 不锈钢、碳化钨、钛合金、哈氏合金
   - 塑料类: 尼龙、PTFE、PU、HDPE
   - 天然材料: 玛瑙

2. **研磨介质 (Grinding Media)**
   - 陶瓷球/珠: 氧化铝、氧化锆
   - 金属球: 不锈钢、碳化钨

3. **研磨设备 (Equipment)**
   - 行星球磨机
   - 振动磨机
   - 辊磨机

4. **配件 (Accessories)**
   - 盖子、密封圈、夹具等

### 关键技术参数
- 工作容积: 50ml - 2000ml
- 纯度: 99.5% - 99.9%
- 硬度: 7.0 - 9.5 Mohs
- 最高温度: 80°C - 1800°C

### 客户类型
- 企业客户 (Enterprise)
- 研究机构 (Institution)
- 分销商 (Distributor)
