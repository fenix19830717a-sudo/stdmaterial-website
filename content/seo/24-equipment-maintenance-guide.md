## Daily Maintenance Tasks

### Pre-Operation Checklist

- [ ] Inspect jar clamps for tightness
- [ ] Check jar seals for damage
- [ ] Verify grinding media condition
- [ ] Clean jar exterior and clamp areas
- [ ] Check emergency stop functionality
- [ ] Review previous shift log

### Post-Operation Procedures

| Task | Time Required | Purpose |
|------|---------------|---------|
| Remove and clean jars | 10 min | Prevent cross-contamination |
| Wipe down machine exterior | 5 min | Maintain appearance, spot issues |
| Empty collection tray | 2 min | Prevent overflow |
| Log operating hours | 1 min | Track maintenance intervals |
| Check for unusual noises | 2 min | Early problem detection |

## Weekly Inspection Protocol

### Mechanical Components

| Component | Inspection | Acceptable Condition | Action if Failed |
|-----------|-----------|---------------------|------------------|
| Drive belts | Visual + tension | No cracks, proper deflection (10mm) | Replace/tension |
| Bearings | Listen + feel | Smooth rotation, no heat | Lubricate/replace |
| Pulleys | Visual | No wear grooves, aligned | Replace/realign |
| Couplings | Visual | Intact, no play | Replace |

### Electrical Systems

**Checklist:**
- [ ] Control panel display functions
- [ ] All buttons and switches responsive
- [ ] RPM readout accurate (verify with tachometer)
- [ ] Timer functions correctly
- [ ] No loose connections
- [ ] Grounding intact

### Safety Systems

| Safety Feature | Test Method | Frequency |
|----------------|-------------|-----------|
| Emergency stop | Press and reset | Weekly |
| Door interlock | Attempt start with door open | Weekly |
| Overload protection | Review trip history | Weekly |
| Vibration sensor | Check calibration | Monthly |

## Monthly Deep Maintenance

### Lubrication Schedule

| Component | Lubricant Type | Interval | Quantity |
|-----------|---------------|----------|----------|
| Main bearings | NLGI #2 grease | Monthly | 10g per bearing |
| Motor bearings | Manufacturer spec | Quarterly | Per manual |
| Gearbox | ISO VG 68 oil | 6 months | Fill to sight glass |
| Guide rails | Light machine oil | Monthly | Thin film |

### Belt Tension Adjustment

**Procedure:**
1. Power off and lock out
2. Measure deflection at center span
3. Target: 10-15mm deflection at 10N force
4. Adjust motor mount position
5. Verify parallel alignment
6. Run test and recheck

### Vibration Analysis

**Acceptable Vibration Levels:**

| Measurement Point | Good | Acceptable | Unacceptable |
|-------------------|------|------------|--------------|
| Motor bearing | <2 mm/s | 2-4 mm/s | >4 mm/s |
| Main spindle | <3 mm/s | 3-5 mm/s | >5 mm/s |
| Machine base | <1 mm/s | 1-2 mm/s | >2 mm/s |

## Component-Specific Care

### Grinding Jars

**Cleaning Protocol:**

| Jar Material | Cleaning Method | Drying |
|--------------|-----------------|--------|
| Stainless steel | Soap and water, acetone rinse | Air dry |
| Tungsten carbide | Alcohol wipe only | Air dry |
| Zirconia | Deionized water, ethanol | Oven 80°C |
| Agate | Water only, no detergents | Air dry |

**Inspection for Damage:**
- Check inner surface for scratches (>0.5mm deep = replace)
- Verify O-ring groove integrity
- Inspect threads for damage
- Check wall thickness (replace if <90% original)

### Grinding Media

**Wear Assessment:**

| Media Type | Replace When | Typical Lifespan |
|------------|--------------|------------------|
| Steel balls | Diameter <90% | 500-1000 hours |
| Zirconia balls | Diameter <95% | 2000-5000 hours |
| Tungsten carbide | Diameter <95% | 3000-8000 hours |
| Alumina balls | Diameter <90% | 1000-2000 hours |

### Drive System

**Motor Maintenance:**
- Check winding resistance annually
- Verify cooling fan operation
- Inspect terminal connections
- Measure current draw vs. nameplate

**Bearing Replacement Indicators:**
- Temperature >80°C during operation
- Noise level increase >10dB
- Vibration increase >50%
- Visible grease leakage

## Common Issues & Solutions

### Problem Diagnostic Matrix

| Symptom | Likely Cause | Solution |
|---------|--------------|----------|
| Excessive vibration | Unbalanced jars | Redistribute load |
| | Worn bearings | Replace bearings |
| | Loose mounting | Tighten bolts |
| Overheating | Insufficient cooling | Clean vents, check fan |
| | Overloading | Reduce batch size |
| | Worn bearings | Lubricate/replace |
| Unusual noise | Loose components | Inspect and tighten |
| | Damaged media | Replace grinding balls |
| | Bearing failure | Emergency stop, inspect |
| RPM fluctuation | Belt slipping | Tension/replace belts |
| | Power supply issue | Check voltage stability |
| | Controller fault | Service electronics |

### Emergency Procedures

**If Equipment Shuts Down Unexpectedly:**

1. Do not restart immediately
2. Check for obvious causes (overload, jam)
3. Inspect for damage or leaks
4. Allow cooling if overheated
5. Document incident
6. Contact service if problem recurs

## Safety Considerations

### Lockout/Tagout (LOTO) Procedure

**Before Any Maintenance:**

1. Notify affected personnel
2. Shut down equipment properly
3. Isolate energy sources
4. Apply lockout devices
5. Verify zero energy state
6. Proceed with maintenance

### Personal Protective Equipment

| Task | Required PPE |
|------|--------------|
| General cleaning | Safety glasses, gloves |
| Belt work | Safety glasses, gloves, closed-toe shoes |
| Bearing service | Safety glasses, cut-resistant gloves |
| Electrical work | Insulated tools, voltage tester |
| Heavy lifting | Steel-toe boots, back support |

### Chemical Safety

When cleaning jars and media:
- Use appropriate solvents for material
- Work in ventilated area
- Dispose of waste properly
- Reference SDS for all chemicals

## FAQ

**Q: How often should I replace grinding media?**
A: Monitor diameter regularly. Replace when reduced by >5% for precision work or >10% for general use.

**Q: What causes jars to crack?**
A: Thermal shock, over-tightening, impact damage, or material fatigue. Always allow jars to cool gradually.

**Q: Can I use compressed air to clean the machine?**
A: Only on exterior surfaces. Never blow air into electrical enclosures or bearing housings.

**Q: How do I know if bearings need replacement?**
A: Warning signs: increased noise, heat, vibration, or visible damage. When in doubt, replace.

**Q: What's the warranty coverage for maintenance?**
A: Warranty typically covers defects, not wear items. Refer to your specific warranty terms.
