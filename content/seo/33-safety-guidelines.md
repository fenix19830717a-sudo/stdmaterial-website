## Operating Procedures

### Pre-Operation Checklist

**Before Starting the Mill:**

- [ ] Inspect equipment for damage
- [ ] Verify all guards in place
- [ ] Check emergency stop function
- [ ] Ensure proper ventilation
- [ ] Don required PPE
- [ ] Review material SDS
- [ ] Check jar clamp tightness
- [ ] Verify timer settings

### Standard Operating Procedure (SOP)

**Step-by-Step Process:**

1. **Preparation (5 min)**
   - Review job safety analysis
   - Don PPE
   - Prepare materials

2. **Loading (5 min)**
   - Add media to jar first
   - Add sample material
   - Seal jar properly
   - Inspect seal integrity

3. **Setup (3 min)**
   - Secure jar in mill
   - Set parameters
   - Verify clear area

4. **Operation**
   - Start mill from safe position
   - Monitor for 5 minutes
   - Periodic checks during run

5. **Shutdown (5 min)**
   - Wait for complete stop
   - Remove jar carefully
   - Open in safe area
   - Clean equipment

### Prohibited Practices

**NEVER:**
- Bypass safety interlocks
- Open mill while running
- Use damaged jars
- Grind incompatible materials together
- Leave running mill unattended (without auto-stop)
- Use excessive force on clamps
- Grind without proper PPE

## Material-Specific Hazards

### Hazardous Material Categories

**Category A - High Hazard:**

| Material | Primary Hazard | Special Precautions |
|----------|----------------|---------------------|
| Pyrophoric metals | Fire | Inert atmosphere only |
| Explosive compounds | Explosion | Prohibited from grinding |
| Toxic powders | Inhalation | HEPA filtration, suit |
| Radioactive materials | Radiation | Licensed facility only |
| Strong oxidizers | Fire/explosion | Isolate from organics |

**Category B - Moderate Hazard:**

| Material | Primary Hazard | Precautions |
|----------|----------------|-------------|
| Heavy metals | Toxicity | N95, gloves, cleanup |
| Nanopowders | Respiratory | P100, containment |
| Reactive chemicals | Burns/corrosion | Face shield, apron |
| Crystalline silica | Silicosis | Wet methods preferred |

**Category C - Low Hazard:**

| Material | Precautions |
|----------|-------------|
| Common minerals | Standard PPE |
| Ceramics | Dust mask |
| Polymers | Standard PPE |

### Compatibility Chart

**Materials That Must NOT Be Ground Together:**

| Material A | Material B | Risk |
|------------|------------|------|
| Oxidizers | Organics | Fire/explosion |
| Acids | Bases | Violent reaction |
| Water | Alkali metals | Fire/explosion |
| Chlorates | Sulfur | Explosion |
| Peroxides | Combustibles | Fire |

## Emergency Procedures

### Emergency Response Framework

```
EMERGENCY RESPONSE HIERARCHY

Level 1 - Minor Incident
    ↓
Self-treatment / First aid
Clean up / Contain
Report to supervisor

Level 2 - Moderate Incident
    ↓
Activate emergency stop
Evacuate immediate area
Notify emergency coordinator
Document incident

Level 3 - Major Emergency
    ↓
Activate building alarm
Full evacuation
Call emergency services
Activate incident command
```

### Specific Emergency Procedures

**Fire:**
1. Activate nearest fire alarm
2. Use fire extinguisher if safe (PASS method)
3. Evacuate if fire grows
4. Account for personnel
5. Do not re-enter

**Chemical Spill:**
1. Alert nearby personnel
2. Don appropriate PPE
3. Contain spill (if safe)
4. Use spill kit materials
5. Dispose per regulations
6. Document incident

**Medical Emergency:**
1. Call emergency services
2. Provide first aid if trained
3. Do not move injured person
4. Clear access for responders
5. Document incident

**Equipment Malfunction:**
1. Press emergency stop
2. Disconnect power
3. Secure area
4. Notify maintenance
5. Do not attempt repair

## FAQ

**Q: Can I operate the mill alone?**
A: Yes, for routine operations with proper training. Never alone when handling hazardous materials.

**Q: How do I know if my material is hazardous?**
A: Check the Safety Data Sheet (SDS) for hazard classifications and handling requirements.

**Q: What should I do if I inhale grinding dust?**
A: Remove to fresh air immediately. If symptoms persist, seek medical attention. Report to supervisor.

**Q: Are there materials that should never be ground?**
A: Yes - explosives, shock-sensitive materials, and incompatible combinations. Consult safety officer.

**Q: How often should safety training be renewed?**
A: Annually for standard operations, more frequently for high-hazard materials or after incidents.
