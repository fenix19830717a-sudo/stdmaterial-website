# 模拟器智能推荐与UI/UX改进 - 产品需求文档

## 概述
- **摘要**：对研磨模拟器进行全面升级，包括智能推荐系统的公式设定和产品参数匹配，以及UI/UX改进，以提升用户体验和品牌一致性。
- **目的**：通过智能推荐系统帮助用户快速找到合适的研磨设备组合，并通过优化的界面设计提升操作体验。
- **目标用户**：使用研磨模拟器的专业用户和潜在客户。

## 目标
- 实现基于产品参数和公式的智能推荐系统
- 确保推荐逻辑准确匹配用户需求
- 优化产品数据结构，增加必要的参数
- 实现符合品牌视觉规范的颜色方案
- 确保所有颜色组合达到WCAG AA级可访问性标准
- 实现流畅的动态效果，包括悬停、点击、加载和过渡状态
- 优化动画性能，确保60fps的稳定帧率
- 强化重点元素，提升用户操作引导

## 非目标（范围外）
- 重写整个模拟器的核心逻辑
- 改变模拟器的基本功能流程
- 添加新的业务功能
- 重构现有的整个代码架构

## 背景与上下文
- 现有的模拟器界面已经完成了从多步骤到单页面的转换
- 智能推荐系统需要基于产品参数和公式进行推荐匹配
- 产品数据结构需要分析和优化，确保包含推荐所需的所有参数
- 目前需要提升界面的视觉效果和用户交互体验
- 品牌视觉规范已经存在，包含具体的颜色值和设计系统

## 功能需求
- **FR-1**：分析现有产品数据结构，确定推荐所需的参数
- **FR-2**：设计智能推荐的公式和匹配逻辑
- **FR-3**：优化产品数据结构，增加必要的参数
- **FR-4**：实现基于产品参数和公式的智能推荐系统
- **FR-5**：实现符合品牌视觉规范的颜色方案，包括主色、辅助色与强调色的选择与应用
- **FR-6**：建立明确的功能类型色彩体系，包括主要操作按钮、次要操作按钮、危险操作按钮、禁用状态按钮和语义化状态颜色
- **FR-7**：实现悬停状态效果，包括颜色深浅变化、微小缩放或阴影增强
- **FR-8**：实现点击反馈效果，包括颜色加深、轻微内陷效果
- **FR-9**：为操作按钮实现加载动画
- **FR-10**：为状态变化实现平滑过渡效果
- **FR-11**：为键盘导航用户设计清晰可见的焦点指示器

## 非功能需求
- **NFR-1**：确保所有颜色组合达到WCAG AA级可访问性标准，文本与背景对比度不低于4.5:1
- **NFR-2**：所有动效持续时间严格控制在200-300毫秒区间
- **NFR-3**：确保动画帧率稳定在60fps，避免引起页面卡顿或性能问题
- **NFR-4**：建立统一的动效规范，确保相似元素行为一致
- **NFR-5**：避免过度动画效果，防止分散用户注意力或造成操作延迟
- **NFR-6**：使用CSS transitions/animations或轻量级JS动画库实现，优先考虑硬件加速属性
- **NFR-7**：智能推荐系统响应时间不超过500ms
- **NFR-8**：推荐结果准确率不低于80%

- **业务**：严格遵循品牌视觉规范，确保品牌一致性
- **依赖**：依赖现有的模拟器HTML结构和JavaScript逻辑
- **数据**：基于现有的产品数据结构进行优化

## 智能推荐公式设计
### 推荐模型选择
- **处理能力匹配**：根据用户输入的Processing Capacity，筛选处理能力相近的设备
- **进料粒度匹配**：根据用户输入的Initial Particle Size，筛选进料粒度大于等于用户输入的设备
- **物料类型匹配**：根据用户输入的Material Type，筛选适合该物料的设备
- **推荐分数计算**：
  ```
  推荐分数 = (处理能力匹配度 × 0.4) + (进料粒度匹配度 × 0.3) + (物料类型匹配度 × 0.3)
  ```
  - 处理能力匹配度：用户输入处理能力与设备处理能力的接近程度
  - 进料粒度匹配度：设备进料粒度与用户输入初始粒度的匹配程度
  - 物料类型匹配度：设备对用户输入物料类型的适合程度

### 推荐球磨罐选择
- **根据模型兼容性**：选择与推荐模型兼容的球磨罐
- **根据物料类型**：选择适合用户输入物料类型的球磨罐材质
- **根据处理能力**：选择容量与推荐处理能力匹配的球磨罐

### 推荐研磨球选择
- **根据球磨罐材质**：选择与球磨罐材质匹配的研磨球
- **根据物料类型**：选择适合用户输入物料类型的研磨球
- **根据目标粒度**：选择适合用户目标粒度的研磨球尺寸

### 估计批次容量计算
- **基于设备处理能力**：根据推荐模型的处理能力
- **基于球磨罐容量**：根据推荐球磨罐的容量
- **基于研磨球填充率**：考虑研磨球的填充率（通常为球磨罐容量的30-50%）
  ```
  估计批次容量 = 球磨罐容量 × 填充率 × 物料密度修正系数
  ```

## 验收标准

### AC-1：智能推荐系统
- **给定**：用户输入Material Type、Processing Capacity和Initial Particle Size
- **当**：用户提交输入
- **则**：系统基于产品参数和公式推荐合适的设备组合，包括Recommended Model、Recommended Jar、Recommended Balls和Estimated Batch Capacity
- **验证**：`programmatic`
- **备注**：推荐结果应与用户需求匹配，响应时间不超过500ms

### AC-2：产品参数匹配
- **给定**：产品数据结构
- **当**：系统进行推荐
- **则**：系统能够正确匹配产品参数，包括处理能力、进料粒度、出料粒度等
- **验证**：`programmatic`
- **备注**：需要确保产品数据结构包含所有必要的参数

### AC-3：颜色方案设计
- **给定**：模拟器界面
- **当**：应用新的颜色方案
- **则**：界面颜色符合品牌视觉规范，所有颜色组合达到WCAG AA级可访问性标准
- **验证**：`human-judgment`
- **备注**：需要验证文本与背景对比度不低于4.5:1

### AC-4：功能类型色彩体系
- **给定**：模拟器界面的按钮和状态元素
- **当**：应用功能类型色彩体系
- **则**：主要操作按钮使用品牌主色，次要操作按钮使用中性色系，危险操作按钮使用警示色系，禁用状态按钮降低饱和度与对比度，信息/成功/警告/错误状态使用对应的语义化颜色
- **验证**：`human-judgment`

### AC-5：悬停状态效果
- **给定**：按钮和可交互元素
- **当**：用户将鼠标悬停在元素上
- **则**：元素出现颜色深浅变化(±10-15%亮度)、微小缩放(1.02-1.05倍)或阴影增强效果
- **验证**：`human-judgment`

### AC-6：点击反馈效果
- **给定**：按钮和可交互元素
- **当**：用户点击元素
- **则**：元素出现明确的按压状态（如颜色加深、轻微内陷效果），反馈时间不超过100ms
- **验证**：`human-judgment`

### AC-7：加载状态动画
- **给定**：操作按钮
- **当**：按钮被点击后执行操作
- **则**：按钮显示简洁的加载动画（如旋转图标、进度条或骨架屏）
- **验证**：`human-judgment`

### AC-8：过渡效果
- **给定**：状态变化的元素
- **当**：元素状态发生变化
- **则**：元素实现平滑过渡，包括颜色、大小、位置等属性的变化动画
- **验证**：`human-judgment`

### AC-9：焦点状态指示器
- **给定**：可交互元素
- **当**：用户使用键盘导航聚焦到元素
- **则**：元素显示清晰可见的焦点指示器
- **验证**：`human-judgment`

### AC-10：动画性能
- **给定**：模拟器界面
- **当**：执行动画效果
- **则**：动画帧率稳定在60fps，动效持续时间在200-300毫秒区间
- **验证**：`programmatic`

### AC-11：重点元素强化
- **给定**：关键操作按钮和重要选项元素
- **当**：查看界面
- **则**：关键操作按钮和重要选项元素应用更鲜明的颜色对比，通过颜色饱和度、亮度或尺寸差异突出核心功能入口，确保重点元素在视觉层级中处于突出位置
- **验证**：`human-judgment`
