# 智能推荐系统技术方案

## 1. 产品数据结构分析

通过分析 `products.json` 文件，我们可以看到产品数据包含以下核心结构：

### 1.1 基本信息
- `id`: 产品唯一标识符
- `name`: 产品名称（中文）
- `nameEn`: 产品名称（英文）
- `category`: 产品类别（英文）
- `subcategory`: 产品子类别（英文）
- `categoryCn`: 产品类别（中文）
- `description`: 产品描述（中文）
- `descriptionEn`: 产品描述（英文）
- `price`: 价格
- `stock`: 库存状态
- `images`: 产品图片

### 1.2 产品参数
产品参数存储在 `specifications` 对象中，包含以下关键参数：

#### 1.2.1 处理能力相关参数
- `处理量`/`产能`/`生产量`/`生产能力`：表示设备的处理能力
- `研磨容积`：表示研磨设备的容量
- `磨矿量`/`给矿量`：表示矿石处理量

#### 1.2.2 粒度相关参数
- `进料粒度`/`入料粒度`/`给料粒度`：表示原料的最大尺寸
- `出料粒度`/`粉碎粒度`/`粉碎细度`：表示处理后产品的尺寸

#### 1.2.3 其他技术参数
- `转速`/`动颚转速`/`瓷辊转速`：设备运行速度
- `功率`：设备功率
- `工作温度`：工作环境温度
- `使用压力`：使用压力要求

### 1.3 数据问题分析
1. **参数命名不统一**：同一参数有多种命名方式（如处理能力有多种表述）
2. **单位不一致**：不同产品的参数单位可能不同
3. **数据格式不规范**：部分参数值包含特殊字符（如 `<`、`>`）
4. **缺失必要参数**：部分产品缺少关键参数

## 2. 智能推荐系统技术方案

### 2.2 数据预处理模块

#### 2.2.1 数据清洗与标准化
1. **参数名称标准化**：将不同命名的同一参数统一命名
   - 处理能力：统一为 `processingCapacity`
   - 进料粒度：统一为 `feedSize`
   - 出料粒度：统一为 `outputSize`
   - 功率：统一为 `power`
   - 转速：统一为 `speed`

2. **单位标准化**：将不同单位的参数转换为标准单位
   - 处理能力：统一为 kg/h
   - 粒度：统一为 mm
   - 功率：统一为 kW

3. **数据格式规范化**：
   - 去除特殊字符（如 `<`、`>`）
   - 处理范围值（如 "1~10kg/h" 转换为最小值和最大值）
   - 填充缺失值

#### 2.2.2 特征提取
从产品数据中提取以下特征用于推荐：
- 类别特征：`category`、`subcategory`
- 技术特征：`processingCapacity`、`feedSize`、`outputSize`、`power`、`speed`
- 应用特征：`applications`

### 2.3 推荐算法

采用混合推荐算法，结合以下方法：

#### 2.3.1 基于内容的推荐
- **相似度计算**：使用余弦相似度计算产品间的相似性
- **特征权重**：根据用户查询的重要性分配不同权重

#### 2.3.2 基于规则的推荐
- **硬约束过滤**：根据用户指定的参数范围过滤产品
- **优先级排序**：根据用户关注的参数优先级排序

#### 2.3.3 协同过滤（可选）
- **用户行为分析**：分析用户浏览和查询历史
- **相似用户推荐**：基于相似用户的行为推荐产品

### 2.4 API接口设计

#### 2.4.1 推荐接口

**接口地址**：`/api/recommend`

**请求方法**：POST

**请求参数**：
```json
{
  "category": "Crushing Series", // 产品类别
  "processingCapacity": {
    "min": 10, // 最小处理能力 (kg/h)
    "max": 100 // 最大处理能力 (kg/h)
  },
  "feedSize": {
    "max": 10 // 最大进料粒度 (mm)
  },
  "outputSize": {
    "max": 0.1 // 最大出料粒度 (mm)
  },
  "applications": ["Mining", "Chemical Industry"], // 应用领域
  "priority": ["processingCapacity", "outputSize"] // 优先级
}
```

**响应格式**：
```json
{
  "status": "success",
  "data": {
    "recommendedProducts": [
      {
        "id": "CR-不锈钢-442",
        "name": "不锈钢振动粉碎机",
        "nameEn": "Stainless Steel Vibration Crusher",
        "category": "Crushing Series",
        "categoryCn": "破碎系列",
        "description": "Professional 不锈钢振动粉碎机 designed for industrial and laboratory applications.",
        "specifications": {
          "processingCapacity": "1~10kg/h",
          "feedSize": "≤3mm",
          "outputSize": "120～1200 目（44～3.1um）"
        },
        "score": 0.95, // 推荐得分
        "images": ["assets/images/products/gm-ss-8.jpg"]
      },
      // 更多推荐产品...
    ],
    "total": 5
  }
}
```
