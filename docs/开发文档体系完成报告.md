#### 10. PRODUCT_ID_SEPARATION.md
**标题**: 产品 ID 分离实施规范  
**优先级**: P0 - 关键要求

**核心内容**:
1. 字段定义
   - productId: 内部标识符 (PROD-00123)
   - sku: 库存单位 (PM-400-STD)
   - name: 产品名称 (Planetary Ball Mill PM-400)
   - slug: URL 友好名称 (planetary-ball-mill-pm-400)

2. 生成规则
   - productId: PROD- + 5 位数字
   - sku: 类型 - 型号 - 变体
   - name: 产品类型 + 型号 + 特性
   - slug: 小写连字符分隔

3. 使用场景
   - productId: 数据库索引、API 调用
   - sku: 订单、库存、采购
   - name: 网站展示、搜索
   - slug: URL 路由

4. 数据库模型
   - Schema 定义
   - 索引配置
   - 自动生成

5. API 接口
   - 通过 slug 获取 (SEO)
   - 通过 productId 获取 (内部)

6. 前端组件
   - ProductCard 实现
   - 字段显示规范

**字段对比表**: 4 字段详细对比

**常见错误**: 3 种典型错误示例

**代码审查清单**: 12 项检查
