## 🏗️ 项目概述

本项目是湖南盛通达材料科技公司的官方 B2B 网站，采用现代化的技术栈构建，提供研磨设备展示、在线模拟器、客户询盘等功能。

### 核心功能

- 🌐 多语言支持（中文、英文、西班牙文、德文）
- 🛍️ 产品目录与详情展示
- 🔬 研磨模拟器
- 🎯 行业匹配工具
- 💬 在线客服系统
- 📊 管理后台

### 技术特点

- ⚛️ React 19 + TypeScript
- 🚀 Vite 构建工具
- 🎨 TailwindCSS + shadcn-ui
- 🗄️ MongoDB 数据库
- 🌐 Express.js 后端服务
