# 部署指南

本文档详细介绍如何将独立站系统部署到生产环境，包括服务器准备、依赖安装、配置设置、服务启动等步骤。

## 1. 服务器准备

### 1.1 服务器要求
- **操作系统**: Ubuntu 20.04 LTS 或更高版本
- **CPU**: 至少 2 核
- **内存**: 至少 4GB
- **存储空间**: 至少 50GB
- **网络**: 稳定的互联网连接
- **端口**: 80（HTTP）、443（HTTPS）、5000（Flask 开发服务器）

### 1.2 服务器初始化
1. **更新系统**
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```

2. **安装必要的系统包**
   ```bash
   sudo apt install -y git curl wget unzip nginx python3 python3-pip python3-venv
   ```

3. **创建项目目录**
   ```bash
   sudo mkdir -p /var/www/independence-site
   sudo chown -R $USER:$USER /var/www/independence-site
   ```

## 3. 配置 Nginx

### 3.1 创建 Nginx 配置文件
1. **创建配置文件**
   ```bash
   sudo nano /etc/nginx/sites-available/independence-site
   ```

2. **配置 Nginx**
   ```nginx
   server {
       listen 80;
       server_name your-domain.com www.your-domain.com;
       
       location / {
           proxy_pass http://127.0.0.1:5000;
           proxy_set_header Host $host;
           proxy_set_header X-Real-IP $remote_addr;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
           proxy_set_header X-Forwarded-Proto $scheme;
       }
       
       location /assets {
           alias /var/www/independence-site/assets;
           expires 30d;
       }
       
       error_page 404 /404.html;
       location = /404.html {
           root /var/www/independence-site;
       }
   }
   ```

### 3.2 启用站点
1. **创建符号链接**
   ```bash
   sudo ln -s /etc/nginx/sites-available/independence-site /etc/nginx/sites-enabled/
   ```

2. **测试 Nginx 配置**
   ```bash
   sudo nginx -t
   ```

3. **重启 Nginx**
   ```bash
   sudo systemctl restart nginx
   ```

## 4. 配置 Gunicorn

### 4.1 创建 Gunicorn 配置文件
1. **创建配置文件**
   ```bash
   nano gunicorn.conf.py
   ```

2. **配置 Gunicorn**
   ```python
   bind = '127.0.0.1:5000'
   workers = 4
   timeout = 120
   accesslog = 'gunicorn-access.log'
   errorlog = 'gunicorn-error.log'
   ```

### 4.2 创建 Systemd 服务
1. **创建服务文件**
   ```bash
   sudo nano /etc/systemd/system/independence-site.service
   ```

2. **配置服务**
   ```ini
   [Unit]
   Description=Independence Site Flask Application
   After=network.target
   
   [Service]
   User=your-user
   WorkingDirectory=/var/www/independence-site
   Environment="PATH=/var/www/independence-site/venv/bin"
   ExecStart=/var/www/independence-site/venv/bin/gunicorn -c gunicorn.conf.py app:app
   Restart=always
   
   [Install]
   WantedBy=multi-user.target
   ```

3. **启用并启动服务**
   ```bash
   sudo systemctl enable independence-site
   sudo systemctl start independence-site
   ```

4. **查看服务状态**
   ```bash
   sudo systemctl status independence-site
   ```

## 9. 备份策略

### 9.1 定期备份
1. **创建备份脚本**
   ```bash
   nano backup.sh
   ```

2. **编写备份脚本**
   ```bash
   #!/bin/bash
   
   BACKUP_DIR="/var/www/independence-site/backups"
   TIMESTAMP=$(date +"%Y%m%d_%H%M%S")
   
   mkdir -p $BACKUP_DIR/$TIMESTAMP
   
   # 备份数据文件
   cp -r /var/www/independence-site/data $BACKUP_DIR/$TIMESTAMP/
   
   # 备份配置文件
   cp /var/www/independence-site/.env $BACKUP_DIR/$TIMESTAMP/
   
   # 压缩备份
   tar -czf $BACKUP_DIR/${TIMESTAMP}.tar.gz $BACKUP_DIR/$TIMESTAMP/
   
   # 删除临时目录
   rm -rf $BACKUP_DIR/$TIMESTAMP
   
   # 保留最近 7 天的备份
   find $BACKUP_DIR -name "*.tar.gz" -mtime +7 -delete
   
   echo "Backup completed: $BACKUP_DIR/${TIMESTAMP}.tar.gz"
   ```

3. **设置执行权限**
   ```bash
   chmod +x backup.sh
   ```

4. **添加到 crontab**
   ```bash
   crontab -e
   ```

   ```
   # 每天凌晨 2 点执行备份
   0 2 * * * /var/www/independence-site/backup.sh
   ```

## 10. 故障排除

### 10.1 常见问题

#### 10.1.1 服务启动失败
- **检查服务状态**
  ```bash
  sudo systemctl status independence-site
  ```

- **查看错误日志**
  ```bash
  sudo journalctl -u independence-site
  ```

- **检查端口占用**
  ```bash
  netstat -tuln | grep 5000
  ```

#### 10.1.2 访问网站显示 502 Bad Gateway
- **检查 Gunicorn 服务**
  ```bash
  sudo systemctl status independence-site
  ```

- **检查 Nginx 配置**
  ```bash
  sudo nginx -t
  ```

- **查看 Nginx 错误日志**
  ```bash
  tail -f /var/log/nginx/error.log
  ```

#### 10.1.3 图片上传失败
- **检查目录权限**
  ```bash
  ls -la /var/www/independence-site/assets/images/
  ```

- **检查目录存在**
  ```bash
  mkdir -p /var/www/independence-site/assets/images/products
  ```

- **设置正确权限**
  ```bash
  chmod -R 755 /var/www/independence-site/assets
  ```

#### 10.1.4 数据库连接失败
- **检查数据库服务**
  ```bash
  sudo systemctl status mysql
  ```

- **测试数据库连接**
  ```bash
  mysql -u independence_user -p
  ```

- **检查数据库配置**
  - 确认 .env 文件中的数据库配置正确

## 12. 安全加固

### 12.1 防火墙配置
1. **安装 UFW**
   ```bash
   sudo apt install -y ufw
   ```

2. **配置防火墙**
   ```bash
   sudo ufw default deny incoming
   sudo ufw default allow outgoing
   sudo ufw allow ssh
   sudo ufw allow http
   sudo ufw allow https
   sudo ufw enable
   ```

3. **查看防火墙状态**
   ```bash
   sudo ufw status
   ```

### 12.2 禁用不必要的服务
1. **查看运行中的服务**
   ```bash
   sudo systemctl list-unit-files --type=service --state=enabled
   ```

2. **禁用不必要的服务**
   ```bash
   sudo systemctl disable <service-name>
   ```
