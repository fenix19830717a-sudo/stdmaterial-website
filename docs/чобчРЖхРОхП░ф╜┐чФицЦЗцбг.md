# 管理后台使用文档

本文档详细介绍独立站管理后台的使用方法，包括登录、仪表盘、产品管理、订单管理、客户管理、内容管理等功能模块的操作指南。

## 1. 登录系统

### 1.1 访问登录页面
- 打开浏览器，输入管理后台地址：`http://your-domain.com/admin/login.html`
- 或在网站首页点击"管理后台"链接

### 1.2 登录认证
- 输入用户名：`admin`
- 输入密码：`admin123`
- 点击"登录"按钮
- 登录成功后，系统会自动跳转到仪表盘页面

## 2. 仪表盘

### 2.1 概览
- 登录后首先进入仪表盘页面，显示系统的关键数据和统计信息
- 包括订单统计、客户统计、产品统计等关键指标
- 提供快速操作按钮，方便用户快速访问常用功能

### 2.2 数据统计
- **订单统计**：显示今日订单数、本月订单数、总订单数、订单总金额
- **客户统计**：显示新增客户数、总客户数、活跃客户数
- **产品统计**：显示产品总数、库存状态、热销产品
- **销售趋势**：以图表形式展示销售趋势

### 2.3 快捷操作
- 快速添加产品
- 快速查看待处理订单
- 快速查看新客户
- 快速访问系统设置

## 3. 产品管理

### 3.1 产品列表
- 在左侧导航栏点击"产品管理"进入产品列表页面
- 显示所有产品的基本信息，包括产品名称、分类、价格、库存等
- 支持按名称、分类、价格等条件筛选产品
- 支持按不同字段排序产品
- 支持批量操作，如批量删除、批量修改状态

### 3.2 添加产品
- 点击"添加产品"按钮进入添加产品页面
- 填写产品基本信息：
  - 产品名称（必填）
  - 分类（必填）
  - 子分类
  - 型号
  - 材质
  - 价格
  - 库存
  - 最小起订量
  - 交货期
- 上传产品图片：点击"上传图片"按钮，选择本地图片文件
- 填写产品详情：
  - 产品描述
  - 技术详情
  - 产品特点
  - 应用领域
  - 认证信息
  - 关键词
- 点击"保存"按钮完成产品添加

### 3.3 编辑产品
- 在产品列表中点击产品名称或"编辑"按钮进入编辑页面
- 修改产品信息后点击"保存"按钮
- 系统会自动记录产品版本历史

### 3.4 删除产品
- 在产品列表中选择要删除的产品
- 点击"删除"按钮
- 确认删除操作
- 产品将被永久删除

### 3.5 产品分类管理
- 在产品管理页面点击"分类管理"标签
- 查看所有产品分类和子分类
- 支持添加、编辑、删除分类

### 3.6 产品搜索
- 在产品列表页面的搜索框中输入关键词
- 支持按产品名称、描述、分类等进行搜索
- 点击"搜索"按钮或按回车键执行搜索

## 4. 订单管理

### 4.1 订单列表
- 在左侧导航栏点击"订单管理"进入订单列表页面
- 显示所有订单的基本信息，包括订单号、客户名称、订单日期、状态、总金额等
- 支持按订单号、客户名称、状态等条件筛选订单
- 支持按不同字段排序订单
- 支持批量操作，如批量修改状态、批量删除

### 4.2 订单详情
- 在订单列表中点击订单号进入订单详情页面
- 查看订单的详细信息，包括客户信息、商品信息、价格信息、物流信息等
- 支持修改订单状态、支付状态、物流信息等

### 4.3 添加订单
- 点击"添加订单"按钮进入添加订单页面
- 填写订单基本信息：
  - 客户名称（必填）
  - 客户ID（可选）
  - 订单日期
  - 订单状态
  - 支付方式
  - 支付状态
  - 物流单号
  - 收货地址
  - 备注
- 添加订单商品：
  - 点击"添加商品"按钮
  - 选择商品
  - 输入数量和单价
  - 系统自动计算小计
- 系统自动计算订单总金额
- 点击"保存"按钮完成订单添加

### 4.4 编辑订单
- 在订单列表中点击"编辑"按钮进入编辑页面
- 修改订单信息后点击"保存"按钮
- 系统会自动记录订单版本历史

### 4.5 删除订单
- 在订单列表中选择要删除的订单
- 点击"删除"按钮
- 确认删除操作
- 订单将被永久删除

### 4.6 订单统计
- 在订单管理页面点击"统计"标签
- 查看订单统计数据，包括：
  - 订单数量统计
  - 销售金额统计
  - 订单状态分布
  - 销售趋势

### 4.7 导出订单
- 在订单列表页面点击"导出"按钮
- 系统会生成CSV格式的订单数据文件
- 浏览器会自动下载该文件

## 12. 常见问题与解决方案

### 12.1 登录失败
- 检查用户名和密码是否正确
- 确认网络连接正常
- 如忘记密码，联系技术支持重置

### 12.2 上传图片失败
- 检查图片格式是否为PNG、JPG、JPEG或GIF
- 检查图片大小是否过大
- 确认服务器存储空间是否充足

### 12.3 数据保存失败
- 检查网络连接是否正常
- 确认必填字段是否已填写
- 查看系统日志了解具体错误信息

### 12.4 系统响应缓慢
- 检查服务器资源使用情况
- 清理浏览器缓存
- 如问题持续，联系技术支持

### 12.5 备份失败
- 检查服务器存储空间是否充足
- 确认备份目录权限是否正确
- 查看系统日志了解具体错误信息

## 13. 安全注意事项

### 13.1 账户安全
- 定期修改管理员密码
- 不要与他人共享管理员账户
- 使用强密码，包含字母、数字和特殊字符

### 13.2 数据安全
- 定期创建数据备份
- 不要在公共网络上使用管理后台
- 及时更新系统和依赖库

### 13.3 操作安全
- 在执行批量操作前，确认操作对象
- 在删除数据前，确认是否需要备份
- 不要随意修改系统设置
